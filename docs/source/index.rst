Welcome to FAA NASR Data Processor's documentation!
===================================

FAA NASR Data Processor (FNDP) is a program designed to produce various custom outputs from the NASR data for the MyFlighRoute project.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development. Feature suggestions are welcome via the project's GitHub `issues page <https://github.com/MyFlightRoute/FAA-NASR-Data-Processor/issues>`_.
